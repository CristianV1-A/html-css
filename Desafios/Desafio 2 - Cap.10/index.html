Tags em html

Em html podemos inserir TAGS para inserir os elementos da lista a seguir. Para saber qual TAG deverá ser usada em cada caso passe o mouse em cima de cada elemento para saber o uso correto em cada caso.

  • Paragáfo
  • título
  • Imagens
  • Abreviações
  • Códigos-fonte
  • Listas numeradas
  • Listas com demarcadores
